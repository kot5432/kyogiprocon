# ヘキサうどん AIビジュアライザー

第37回プロコン「ヘキサうどん」のAI判断を可視化する分析ツールです。

## 特徴

- 🎯 **AIの判断が一目で分かる**: 巡回車と補給車の連携、燃料管理、戦略を視覚化
- 📊 **リアルタイム分析**: 燃料ゲージ、スコア推移、ターンログを表示
- 🔮 **未来予測**: 5ターン先までの予定経路を表示
- 🧠 **判断理由**: AIの意思決定プロセスを表示（データがある場合）
- 📈 **スコアグラフ**: 自チームと相手チームのスコア推移を比較

## 使用方法

### 大会の流れに沿った使用方法

ビジュアライザーは第37回プロコン競技部門「ヘキサうどん」の公式フォーマットに対応しています。

#### ステップ1: マップ構成をロード（試合開始前）

1. 「📁 Load Data」セクションの「Map Info JSON」ボタンをクリック
2. 試合開始前に提供されるマップ構成JSONを選択
3. マップが中央に表示されます
4. ロードガイドが「Step 2: Configure Agents」に進みます

**マップ構成フォーマット**:
```json
{
  "startsAt": 1778227200,
  "daySeconds": [5, 5, 5, 10],
  "daySteps": [50, 100, 150, 200],
  "map": {
    "height": 8,
    "width": 8,
    "cells": [...]
  },
  "spots": [...],
  "agents": [4, 12, 20, 28],
  "fuelLimits": 20,
  "players": 8,
  "busyThreshold": 2,
  "jammedThreshold": 4
}
```

#### ステップ2: エージェント種別を設定（試合開始前）

マップデータをロードすると、エージェント設定パネルが表示されます：

**方法A: スライダーで設定**
1. **Patrol Agents（巡回車）**: スライダーで巡回車の数を設定（0-4）
2. **Supply Agents（補給車）**: スライダーで補給車の数を設定（0-4）
3. **Apply Configuration**: 設定を適用ボタンをクリック
4. **Export as JSON**: 大会提出用のJSONファイルをダウンロード

**方法B: JSONファイルをロード**
1. 「Or load Agent Types JSON」で既存のエージェント種別JSONをロード
2. フォーマット: `[0, 1, 0, 1]` （0: 巡回車, 1: 補給車）

**エージェント種別回答フォーマット**:
```json
[0, 1, 0, 1]
```

**注意**: 合計エージェント数はマップで定義された最大数を超えられません。

#### ステップ3: 各日の試合情報をロード（各日開始時）

1. 「📝 Action Data」セクションの「Day Info JSON」を選択
2. 各日開始時に提供される試合情報JSONをロード
3. 自チームのエージェント情報、他チームの情報、渋滞情報が表示されます

**各日開始時の試合情報フォーマット**:
```json
{
  "endsAt": 1778227205,
  "day": 1,
  "agents": [
    {"kind": 0, "pos": 1, "fuel": 20},
    {"kind": 1, "pos": 1, "fuel": 20}
  ],
  "others": [
    {
      "id": 0,
      "agents": [...]
    }
  ],
  "traffics": [
    {"pos": 1, "status": 0}
  ]
}
```

#### ステップ4: 行動計画をロード（各日の回答後）

1. 「Action Plan JSON」で行動計画JSONをロード
2. 各エージェントの行動列が再生されます

**行動計画回答フォーマット**:
```json
[
  [-15],           // エージェント0: 15ステップ待機
  [0, 1, -10]      // エージェント1: 左上→右上→10ステップ待機
]
```

- `-1`以下: 待機（例: `-15` = 15ステップ待機）
- `0-5`: 移動方向（0: 左上, 1: 右上, 2: 右, 3: 右下, 4: 左下, 5: 左）

#### ステップ5: 再生して分析

データをロードした後、以下のコントロールで操作できます：

- **▶ Play**: 自動再生開始
- **⏸ Pause**: 一時停止
- **⏮ Prev**: 1ターン戻る
- **⏭ Next**: 1ターン進む
- **Turnスライダー**: 任意のターンにジャンプ
- **Speedスライダー**: 再生速度調整（1-100）

## 画面構成

### 左サイドバー
- **📁 Load Data**: データファイルのロード
- **🎮 Playback**: 再生コントロール
- **⛽ Fuel Status**: 各エージェントの燃料ゲージ
  - 🔵 巡回車（青）
  - 🟢 補給車（緑）
  - 50%以上: 緑、20-50%: 黄色、20%以下: 赤色、0%: 点滅
- **🧠 AI Decision**: AIの判断理由（データがある場合）

### 中央マップ
- **六角形マップ**: 地形表示
  - 平原（緑）、道（灰色）、山（茶色）、池（水色）
- **スポット**: うどんアイコン（ブランド別に色分け）
- **エージェント**:
  - 巡回車: 青い三角形
  - 補給車: 緑の正方形
- **戦略レイヤー**:
  - 未来経路: 半透明の破線（5ターン先まで）
  - 目的矢印: 現在位置から目標への矢印
- **ヒートマップ**: AI評価値のオーバーレイ（データがある場合）

### 右パネル
- **📊 Score History**: スコア推移グラフ
  - 青: 自チーム
  - 赤: 相手チーム
- **📝 Turn Log**: 最近10ターンのログ

### よくある質問

**Q: マップだけ表示されてエージェントが見えない**
A: Action Plan JSON または Dayデータをロードしてください。

**Q: 燃料ゲージが表示されない**
A: Dayデータをロードするか、アクションデータをロードしてください。

**Q: AI判断理由が表示されない**
A: 拡張形式のデータ（`decisions`フィールド付き）をロードしてください。

**Q: スコアグラフが表示されない**
A: 拡張形式のデータ（`scoreHistory`フィールド付き）をロードしてください。
